# Claims History Guidance (Synthetic)
**Effective Date:** 2026-01-01  
**Version:** v0.1-synthetic

## 1. Purpose
This document defines how prior loss history and coverage continuity affect a homeowners decision. It is synthetic and is used to test multi-document retrieval alongside the eligibility and hazard manuals.

## 2. Loss Frequency
### 2.1 Counting Prior Losses
- Three or more non-catastrophe losses in the prior five years **SHALL** refer.
- Two losses of the same peril (for example, two theft losses) **SHOULD** prompt a cause-of-loss narrative.
- Catastrophe losses from a declared event **MAY** be excluded from frequency counts at underwriter discretion.

### 2.2 Loss Severity
- Any single prior loss over the program's large-loss threshold **MUST** refer with adjuster notes.
- Open or unresolved claims **SHALL** refer until closed.

## 3. Coverage Continuity
### 3.1 Lapse in Coverage
- A lapse in prior homeowners coverage greater than thirty days **SHOULD** prompt a reason-for-lapse question.
- A lapse driven by non-payment **SHALL** refer.

### 3.2 Prior Carrier Non-Renewal
- A prior non-renewal for underwriting reasons **MUST** refer with the non-renewal notice attached.
- A non-renewal for carrier market exit **MAY** be treated as neutral.

## 4. Verification Sources
- Loss history **SHOULD** be corroborated with a third-party loss report (CLUE-style) when available.
- Discrepancies between the application and the loss report **MUST** be reconciled before binding.

## 5. Documentation for the Decision Packet
- The decision packet **SHOULD** summarize loss count, severity, and continuity findings.
- Cited loss-history rules **MUST** appear in the citations list when they drive a referral.
